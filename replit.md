# Zakir Aleem Portfolio

## Overview
A personal portfolio website for Zakir Aleem, built with React, TypeScript, Vite, and Tailwind CSS. The site showcases information about Zakir from Karachi, Pakistan, with an "Available for opportunities" status.

## Recent Changes
- Migrated from Lovable to Replit environment (January 22, 2026)
- Updated Vite config to use port 5000 and allow all hosts for Replit compatibility
- Removed lovable-tagger dependency

## Project Architecture

### Tech Stack
- **Frontend**: React 18 with TypeScript
- **Build Tool**: Vite 5
- **Styling**: Tailwind CSS with shadcn/ui components
- **Animation**: Framer Motion
- **Routing**: React Router DOM v6

### Directory Structure
```
src/
├── components/     # UI components (including shadcn/ui)
├── hooks/          # Custom React hooks
├── lib/            # Utility functions
├── pages/          # Page components
├── test/           # Test files
├── App.tsx         # Main app component
├── main.tsx        # Entry point
└── index.css       # Global styles
```

### Running the Project
- Development: `npm run dev` (runs on port 5000)
- Build: `npm run build`
- Preview: `npm run preview`
- Tests: `npm test`
